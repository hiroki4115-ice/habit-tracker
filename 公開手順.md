# 習慣トラッカーをiPhoneアプリとして使う手順

所要時間は10分ほどです。無料でできます。

## 1. GitHubに公開する

1. https://github.com にアクセスしてログイン(アカウントがなければ無料登録)
2. 右上の「+」→「New repository」
3. Repository name に `habit-tracker` と入力し、**Public** のまま「Create repository」
4. 「uploading an existing file」のリンクをクリック
5. このフォルダの中身(`index.html`、`manifest.json`、`sw.js`、`icon-180.png`、`icon-192.png`、`icon-512.png`)を全部ドラッグ&ドロップ
   - ※ フォルダごとではなく、ファイルを直接アップロードしてください
6. 「Commit changes」をクリック

## 2. GitHub Pagesを有効にする

1. リポジトリの「Settings」タブ →左メニューの「Pages」
2. 「Branch」で `main` を選び「Save」
3. 1〜2分待つと、ページ上部にURLが表示されます
   (`https://ユーザー名.github.io/habit-tracker/` の形式)

## 3. iPhoneのホーム画面に追加する

1. iPhoneの **Safari** で上のURLを開く
2. 画面下の **共有ボタン(□に↑)** をタップ
3. **「ホーム画面に追加」** をタップ →「追加」

これで完了です。ホーム画面の🌱アイコンから起動すると、全画面のアプリとして動きます。一度開けばオフラインでも使えます。

## 補足

- データはiPhoneの中(Safariのストレージ)に保存されます。サーバーには何も送信されません
- **注意**: iPhoneの設定で「Safariの履歴とWebサイトデータを消去」をすると記録も消えます
- アプリを更新したいときは、GitHubで `index.html` を差し替えるだけ。次に開いたときに反映されます
